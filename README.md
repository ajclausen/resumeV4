# Andrew Clausen - Portfolio

My professional portfolio and resume as a **System Engineer & Network Architect**.

This project showcases my professional experience, technical skills, and projects, bridging the gap between physical infrastructure and modern software.

## Tech Stack

- **React 19** (via Vite)
- **TypeScript**
- **Tailwind CSS** (CDN)
- **Lucide React** for icons

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the app:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```
